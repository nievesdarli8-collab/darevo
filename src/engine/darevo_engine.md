# DAREVO ENGINE
Versión: 0.0.1

## Propósito

El DAREVO Engine es el núcleo del videojuego.

Todos los sistemas deben depender del Engine.

El Engine nunca dependerá de un módulo específico.

---

## Responsabilidades

- Crear partidas.
- Administrar salas.
- Administrar jugadores.
- Seleccionar actividades.
- Controlar el flujo de la partida.
- Ejecutar eventos.
- Registrar estadísticas.
- Administrar el consentimiento.
- Coordinar los motores internos.

---

## Motores internos

- Room Engine
- Player Engine
- Activity Engine
- Challenge Engine
- Question Engine
- Consent Engine
- AI Director
- Event Engine
- Audio Engine
- Statistics Engine

---

## Regla principal

Toda nueva funcionalidad debe integrarse al DAREVO Engine antes de estar disponible para los jugadores.
